# Gesture Strike

一个本地摄像头手势 Web 游戏原型。

## 运行

摄像头权限需要安全上下文，建议通过 localhost 打开：

```bash
cd /Users/yuzhilin/Documents/GitHub/aicoding/gesture-game
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## 玩法

- 点击「启动摄像头」并允许浏览器摄像头权限。
- 把手放到右上角摄像头预览的方框内，点击「校准手部」。
- 做手枪手势时，会沿手指方向自动射击敌人。
- 张开手掌并悬停在 A/B 能量核心上，会推进任务进度。
- 摄像头不可用时，可以用鼠标在画布上移动测试射击反馈。

## 识别方式

该原型不依赖联网模型。它会在校准时采样手部肤色，然后在本地 canvas 中做像素分割，估算手部中心、远端指尖方向、手掌填充度和轮廓拉伸度。

受光照、背景肤色物体、摄像头曝光影响，识别可能需要重新校准。真实产品可替换为 MediaPipe Hands 或自训练手势模型。
