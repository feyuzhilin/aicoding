# 废墟邮差

离线纯本地运行的单人卡牌 Roguelite 第一版原型。

## 运行

直接用浏览器打开 `index.html` 即可，不需要安装依赖或启动服务。

## 已实现

- 8 天路线：遭遇、事件、休整、终点战。
- 回合制卡牌战斗：体力、抽牌、护甲、敌方意图、声响压力。
- 30 张基础/升级卡定义，首版牌池会在奖励中逐步加入。
- 8 种敌人和 4 类路线事件。
- 5 个工具遗物。
- 自动保存到 `localStorage`。
- 手动导出 JSON、复制存档文本、从文本或 JSON 文件导入。

## 设计边界

这一版先验证核心循环：路线选择压力、战斗资源管理、包裹完整度和手动存档迁移。暂不做长期解锁、动画、音效和关卡编辑器。
