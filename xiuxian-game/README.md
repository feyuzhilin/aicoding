## 1. 游戏概述

### 1.1 概念与愿景

一款沉浸式的修仙角色扮演网页游戏，玩家将扮演一名资质平凡的修士，从炼气期开始，通过采集灵草、炼制丹药、击败妖兽不断提升修为，最终突破境界，飞升成仙。游戏氛围古朴神秘，融合中国传统修仙文化元素。

### 1.2 核心特点

| 特点 | 说明 |
|------|------|
| 🖥️ **纯前端** | HTML + CSS + JavaScript，无需后端 |
| 💾 **本地存储** | LocalStorage 保存进度，自动存档 |
| 📤 **导入导出** | 支持存档文件迁移 |
| 🌐 **离线运行** | 无需网络，随时修仙 |
| 📱 **响应式** | 支持桌面/平板/手机 |

### 1.3 如何运行

双击 `index.html` 即可在浏览器中运行游戏。

## 3. 核心功能

### 3.1 境界系统

修仙之路共八个境界，每境界多层突破：

| 境界 | 层数 | 说明 |
|------|------|------|
| 炼气期 | 1-9层 | 初始境界，踏入修仙门槛 |
| 筑基期 | 初/中/后期 | 入门，正式成为修士 |
| 结丹期 | 初/中/后期 | 小成，凝结金丹 |
| 元婴期 | 初/中/后期 | 中成，元婴出窍 |
| 化神期 | 初/中/后期 | 大成，化凡为神 |
| 炼虚期 | 初/中/后期 | 半步大能 |
| 大乘期 | 初/中/后期 | 大能，离飞升一步之遥 |
| 渡劫期 | 初/中/后期 | 渡劫飞升 |

### 3.2 修炼系统

**自动修炼**
- 点击"开始修炼"按钮进入修炼状态
- 每秒自动获得修为，效率受灵根加成
- 再次点击或点击"停止修炼"结束

**突破境界**
- 修为达到上限后可尝试突破
- 突破需要一定概率（受灵根和丹药影响）
- 突破成功：境界提升，属性增强
- 突破失败：损失部分修为

**灵根影响**
- 灵根资质影响修炼效率
- 灵根越高，突破成功率越高

### 3.3 战斗系统

**进入战斗**
- 选择妖兽难度（炼气/筑基/结丹）
- 随机遭遇对应境界的妖兽

**战斗操作**
| 操作 | 说明 |
|------|------|
| ⚔️ 攻击 | 普通攻击，造成基于攻击力的伤害 |
| 🛡️ 防御 | 进入防御姿态，减少下次受到的伤害 |
| 📜 技能 | 使用已学习的功法技能 |
| 🏃 逃跑 | 尝试逃离战斗，成功率受敏捷影响 |

**战斗奖励**
- 击败妖兽获得修为
- 概率掉落灵草
- 获得灵石奖励

### 3.4 灵草园系统

**灵草种类**
| 灵草 | 品质 | 生长时间 | 用途 |
|------|------|----------|------|
| 止血草 | 凡品 | 30秒 | 炼制疗伤丹 |
| 灵气草 | 良品 | 45秒 | 炼制聚灵丹 |
| 培元草 | 良品 | 60秒 | 炼制培元丹 |
| 九转草 | 上品 | 120秒 | 炼制破境丹 |
| 灵芝 | 上品 | 180秒 | 炼制高级丹药 |
| 天元草 | 极品 | 300秒 | 炼制仙丹（稀有） |

**操作**
- 点击空闲地块：播种随机灵草
- 点击成熟灵草：采摘入库

### 3.5 炼丹系统

**丹药配方**
| 丹药 | 材料 | 效果 |
|------|------|------|
| 疗伤丹 | 止血草 x2 | 恢复50生命 |
| 聚灵丹 | 灵气草 x2 | 恢复30灵力 |
| 培元丹 | 培元草 x2 + 灵芝 x1 | 获得10修为 |
| 破境丹 | 九转草 x3 + 灵芝 x1 | 突破成功率+20% |

### 3.6 坊市系统

**购买**
- 精钢剑：50灵石，攻击+5
- 护体甲：50灵石，防御+5
- 疗伤丹(中)：30灵石，恢复100生命
- 聚灵丹(中)：25灵石，恢复60灵力

**出售**
- 灵草可出售换取灵石
- 丹药可出售换取灵石

### 3.7 背包系统

**物品分类**
- 灵草：炼丹原材料
- 丹药：使用后获得效果
- 材料：其他消耗品

**使用物品**
- 点击背包中的物品使用
- 丹药立即生效

**功法列表**
| 功法 | 境界要求 | 效果 |
|------|----------|------|

| 火球术 | 筑基期 | 攻击加成，消耗20灵力 |
| 灵治愈术 | 筑基期 | 恢复生命，消耗15灵力 |

**学习功法**
- 进入功法界面查看可学功法
- 满足境界要求即可学习
- 已学功法在战斗中可用
